> A machine learning model that classifies sonar signals to predict whether an underwater object is a Rock or a Mine.

---

## 📌 Overview

This project uses the **Sonar Dataset** to build a binary classification model. Sonar signals bounced off underwater objects are analyzed to distinguish between rocks and metal cylinders (mines).

---

## ✨ Features

- ✅ Binary classification: Rock / Mine
- ✅ Data preprocessing & normalization
- ✅ Exploratory Data Analysis (EDA)
- ✅ Model training & evaluation
- ✅ Accuracy & performance metrics

---

## 📂 Dataset

- **Source:** Sonar Dataset (UCI Machine Learning Repository)
- **Records:** 208 rows × 61 columns
- **Target:** R = Rock, M = Mine
- **Features:** 60 sonar frequency readings

---

## 📊 ML Models Used

- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest Classifier

---
